
# Customer Behavior Analysis

This project analyzes customer data using Python, PostgreSQL, and Power BI.

## 📊 Project Overview
This is an end-to-end data analysis project where raw data is cleaned, analyzed, and visualized to extract meaningful insights about customer behavior.

## 🛠️ Tools Used
- Python (Pandas)
- PostgreSQL
- Power BI

## 🔧 Steps Performed
- Data collection from Kaggle
- Data cleaning using Pandas
- Database creation and SQL analysis
- Dashboard creation in Power BI

## 📈 Key Insights
- Majority of customers are male
- Positive feedback dominates the dataset
- Students are the largest customer group
- Average family size is around 3–4 members

## 📁 Project Files
- Dataset (CSV)
- SQL queries
- Power BI dashboard
- Python script
- Final report
